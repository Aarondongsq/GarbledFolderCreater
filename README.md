# 👋 介绍
一个可以大批量创建乱码文件夹的软件！  

无需繁琐操作，无需填入正则，仅需选择好想生成乱码文件夹的文件夹，然后输入你想生成多少个文件夹，即可一键生成！    

### 说明
有时候我们想存放一些文件，但是这些文件不适合在一些公共的场合出现，需要被隐藏，而加密软件又显得过于费时和麻烦。  

本软件且在解决这个问题，将你所想藏起来的文件放到本软件生成的一堆乱码文件夹中，让其他人无法轻易找到你的文件。  

<br>

**注意：本软件不是隐藏软件！请勿将你的重要文件通过本软件隐藏！**

>[!tip]
> **Windows用户可以把生成的乱码文件夹全选，然后隐藏。**  
> 需要时可以先打开*显示隐藏的文件*，然后进入你隐藏的文件夹中，然后关闭*显示隐藏的文件*

# ✨功能
- [x] 预制好的文件夹生成数量
- [x] 可随机生成文件夹数量 
- [ ] 大小写区分
- [ ] 可选择是否加入数字
- [ ] 子文件夹支持
- [ ] 注入自定义字符
